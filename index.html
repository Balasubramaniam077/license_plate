Number Plate Tracking & Safety System Project

This project was created for a freelancing client.

  • AI-Powered Number Plate Recognition: Developed a system using Python, Flask, and OCR to accurately track and extract vehicle number plates, with automated email notifications and seamless integration into a database.
  • Real-Time Safety Management: Implemented safety features such as helmet detection and pillion rider monitoring to ensure compliance with safety regulations.
  • Embedded System Deployment: Successfully deployed the system on an NVIDIA Jetson Nano, enabling efficient real-time processing and management on an embedded platform.
